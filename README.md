# Git sheet


### To set username and password credentials in github outomatically 
```
git config --global user.name "your username"

git config --global user.password "your password"
```


# Docker sheet

### Build image

```

```

### Run image

```
docker run -it -p 5000:5000 --name=flask 75dd93600ee7
```
